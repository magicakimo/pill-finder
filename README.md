### 為什麼開發這個工具？
由於藥師的日常工作之一是病人的藥品辨識，而目前常見的資料庫無法支援同時搜尋藥品標記、顏色、形狀、刻痕、大小，同時又保有模糊搜尋的空間 (如: 橘色和粉紅色、HS vs SH)

因此，本網頁利用 AI 協助整合衛福部食藥署的開放資料，以常用的刻字、藥名搜尋為主，並搭配上可額外篩選的顏色、形狀、刻痕、大小等，讓藥師在進行藥物辨識時能更快找出對應的藥品。

### 功能亮點
- **便利的刻字搜尋**：考慮到藥錠刻字可能模糊或容易看錯（例如：**數字 0 與 大寫的 O；大寫的 I、小寫的 L 和數字 1**），系統具備模糊比對功能，大幅提升搜尋成功率。
- **180度翻轉也沒問題**：部分藥品的刻字可以 180 度相反閱讀。若食藥署的藥品標記資料翻轉後仍可以閱讀，支援反轉後的搜尋結果 (如：**HS 和 SH**、**110 和 011**)
- **沒有刻痕也可以**：無需輸入文字，只需點選畫面上藥錠的「顏色」或「形狀」，系統就會自動過濾，呈現最接近的結果。
- **同步顯示成分名**：食藥署的藥品外觀資料集不包含成分名，故將藥名和健保用藥品項查詢項目檔做比對，若健保用藥有該藥品，則自動顯示成分名。
- **清晰的大圖核對**：每一種藥物都提供照片對照，點擊圖片即可連結到 TFDA 網站閱讀大圖，方便與手中實體藥物進行細節比對。
- **串聯官方仿單**：點擊藥名可直接跳轉至食藥署官方說明頁面，讓藥物成分與適應症資訊一目了然。

### 如何使用
1. **打開網頁**：使用手機或電腦瀏覽器進入系統。
2. **設定特徵**：在搜尋框輸入藥錠上的刻字，或直接填寫藥品的顏色、形狀、刻痕。
3. **瀏覽結果**：送出結果後，系統會在下方顯示符合條件的藥品。
4. **深入查看**：點擊藥品圖片放大觀察，或點擊中文藥名查看完整的官方藥品仿單。

### 已知問題
- 成分名稱係依資料集之商品名對照健保用藥品項進行比對，非健保品項或特殊品項可能未完整涵蓋。
- STD、SANDOZ 系列的藥品不在資料庫中，請改用其他網站查詢
